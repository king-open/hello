### 基本的命令

1. mkdir demo (新建一个demo的文件名)
2. cd demo (进入demo文件里)
3. touch xxx.py(创建xxx.py)

#### 删除命令

> 必须要在根目录删，通俗的说在这个文件外面删

rm -r demo (这个文件已删除)
